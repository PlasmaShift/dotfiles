=head1 How to set up offline two-factor auth with a yubikey

 git clone git://github.com/firnsy/yubipam.git
 sudo apt-get install libpam0g-dev
 autoreconf -i
 ./configure
 make
 sudo make install
 sudo addgroup yubiauth
 sudo touch /etc/yubikey
 sudo chgrp yubiauth /etc/yubikey /usr/local/sbin/yk_chkpwd
 sudo chmod g+rw /etc/yubikey
 sudo chmod g+s /usr/local/sbin/yk_chkpwd

Put the following line at the top of F</etc/pam.d/common-auth>:

 auth requisite pam_yubikey.so

To add a key:

 ykpasswd -a -u frew -k $AES -o $OTP

B<NOTE:> Do not include the C<h:> in the AES value.

Finally run C<ykvalidate $OTP> and it should identify the configured user.
